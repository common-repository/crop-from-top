== Description ==

Forces image resizing and cropping to occur from the top of an image instead of the center whenever an image is uploaded and resized by WordPress.

== Installation ==

1. Upload the `crop-from-top` folder to the `/wp-content/plugins/` directory
2. Active the Crop From Top plugin through the 'Plugins' menu in WordPress
